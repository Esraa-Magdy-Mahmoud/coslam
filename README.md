# coslam
Cooperative SLAM with ROS.

## Requirements:
  * Ubuntu 16.04.
  * ROS Kinetic.
  * gazebo 7.7.0.
  
  
## Usage :
  * SLAM with One Robot :
   ```
      $ cd coslam/shell_scripts
      $ ./slam_solo.sh
   ```
   * Navigation with One Robot :
   
   ```
      $ cd coslam/shell_scripts
      $ ./nav_robot.sh
   ```
   * Multi-Robot SLAM:
   ```
      $ cd coslam/shell_scripts
      $ ./multi_slam.sh
   ```
   * Multi-Robot Navigation (Still buggy!!)
   
   ```
      $ cd coslam/shell_scripts
      $ ./multi_nav.sh
   ```
